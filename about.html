About Grimoire App

  • Grimoire is a simple, single page app that generates random treasure loot for the Pathfinder Role Playing Game system.
  • It utilizes roll tables and loot tables from the Core Rulebook and the Advanced Player's Guide.
  • Spell lists are taken from the CRB, APG, and additionally Ultimate Magic and several other sources
  • Some randomly determined properties (such as the create type of a Bane weapon) do not have official roll tables. I made up my own roll tables for these values.
  • GMs are highly encouraged to add/remove/disregard any item or part of an item that they want to the random loot drops.

How to Use Grimoire App

  1. Select from the sources on the left hand side
    • Currently only the checkbox and the radio buttons work
    • Marking the checkbox will result in all items being included in the roll
    • Selecting a radio button will result in only that item type being rolled
  2. Select how many randomly generated items you want
    • Reference your Core Rulebook for example of minor, medium, and major loot items
  3. Click "GO"
    • Optional: Select a minimum total value for the treasure rolled up. Grimoire will attempt 10 times before giving up
  • There are several other quick featuers that can be accessed using the buttons on the bottom
